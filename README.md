# EBOOK Interativo com IA 📘

## 📒 Descrição
Este projeto tem como objetivo criar um eBook interativo utilizando a inteligência artificial do ChatGPT como roteirista e a ferramenta Twine para estruturar e desenvolver a narrativa. O resultado é uma experiência de leitura envolvente e dinâmica, onde os leitores podem escolher os caminhos da história e explorar diferentes finais.

## 🤖 Tecnologias Utilizadas
- **ChatGPT**: Utilizado como roteirista para gerar o conteúdo textual do eBook.
- **Twine**: Ferramenta usada para criar histórias interativas e não-lineares.

## 🧐 Processo de Criação
1. **Planejamento da Narrativa**: Inicialmente, foi definido o tema e o enredo principal da história. 
2. **Desenvolvimento do Roteiro com ChatGPT**: Utilizando a IA do ChatGPT, foi gerado o conteúdo textual da história, incluindo diálogos, descrições e ramificações da trama.
3. **Estruturação no Twine**: O conteúdo gerado foi inserido no Twine, onde a história foi organizada em uma estrutura não-linear, permitindo aos leitores fazer escolhas que influenciam o desenrolar dos eventos.
4. **Revisão e Ajustes**: A história foi revisada e ajustes foram feitos para garantir coesão e fluidez na narrativa.

## 🚀 Resultados
O resultado final é um eBook interativo que proporciona uma experiência de leitura única e personalizada. Os leitores têm a liberdade de explorar diferentes caminhos e finais, tornando cada leitura uma nova aventura. Este projeto demonstra o potencial das IAs generativas e das ferramentas de criação interativa para inovar na forma como contamos e consumimos histórias.
